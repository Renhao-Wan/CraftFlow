CraftFlow 使用指南

AI 驱动的智能长文创作与审校平台

快速开始

1. 下载安装

前往下载页面下载适合您操作系统的客户端：

  • Windows: Windows 10 及以上版本
  • macOS: macOS 12 及以上版本 (Apple Silicon)
  • Linux: Ubuntu 20.04 及以上版本

下载完成后，双击安装包按照提示完成安装。

2. 首次启动

启动 CraftFlow 后，您需要配置 LLM 服务：

  1. 点击侧边栏底部的 设置 按钮
  2. 在 LLM 配置 中添加您的 API Key
  3. 选择合适的模型（推荐 GPT-4 或 Claude）

支持所有 OpenAI 兼容格式的 LLM 服务，包括 OpenAI、Anthropic、本地部署的 Ollama 等。

3. 开始创作

配置完成后，您可以：

  • 点击 新建创作 开始智能长文创作
  • 点击 文章润色 对已有内容进行优化

核心功能

智能创作

CraftFlow 采用渐进式织造架构，将长文创作分解为多个阶段：

  1. 大纲生成 — 输入主题，AI 自动生成结构化大纲
  2. 大纲确认 — 您可以审核、修改大纲，确保方向正确
  3. 并发撰写 — AI 按章节并发撰写，突破上下文长度限制
  4. 智能审校 — 自动检查内容质量，确保一致性

使用步骤

  1. 点击侧边栏 新建创作
  2. 输入文章主题和要求
  3. 等待 AI 生成大纲（约 30 秒）
  4. 审核大纲，可直接修改或添加要求
  5. 确认后 AI 开始撰写全文
  6. 实时查看撰写进度

文章润色

提供三档润色模式，满足不同需求：

模式 说明 适用场景
极速格式化 调整格式、修复语法 快速整理文档
专家对抗 Author-Editor 博弈 提升文章质量
事实核查 验证内容准确性 学术/新闻写作

使用步骤

  1. 点击侧边栏 文章润色
  2. 粘贴或上传需要润色的内容
  3. 选择润色模式
  4. 等待处理完成
  5. 查看润色结果，支持导出

人机协作

CraftFlow 在关键节点会暂停执行，等待您的确认：

  • 大纲确认 — AI 生成大纲后等待您审核
  • 中断恢复 — 任务中断后可随时恢复
  • 实时调整 — 在创作过程中提出修改意见

任务管理

所有任务都会自动保存，您可以：

  • 在 任务历史 中查看所有任务
  • 点击任务查看详细状态和结果
  • 恢复中断的任务继续执行
  • 删除不需要的任务记录

常见问题

LLM 相关

Q: 支持哪些 LLM 服务？

A: 支持所有 OpenAI 兼容格式的服务，包括：

  • OpenAI (GPT-4, GPT-4o)
  • Anthropic (Claude 3.5 Sonnet)
  • 本地部署 (Ollama, vLLM)
  • 其他兼容服务 (DeepSeek, Moonshot 等)

Q: 如何配置本地 LLM？

A: 在设置中添加 LLM 配置：

  1. API Base URL 填写本地地址，如 http://localhost:11434/v1
  2. API Key 可随意填写（如 ollama）
  3. 填写模型名称，如 llama3

Q: 创作任务很慢怎么办？

A: 长文创作需要多轮 LLM 调用，通常需要 3-10 分钟。您可以：

  • 使用更快的模型（如 GPT-4o-mini）
  • 减少文章章节数量
  • 在设置中调整并发数

任务相关

Q: 任务中断了怎么办？

A: 任务中断可能是因为：

  • 网络连接问题
  • LLM 服务超时
  • 浏览器关闭

您可以在 任务历史 中找到中断的任务，点击 恢复 继续执行。

Q: 如何导出创作结果？

A: 在任务详情页面，点击 导出 按钮，支持：

  • Markdown 格式
  • PDF 格式
  • 复制到剪贴板

Q: 任务数据保存在哪里？

A:

  • 桌面版: 数据保存在本地 %APPDATA%/CraftFlow/ 目录
  • Web 版: 数据保存在服务器，通过账号同步

技术相关

Q: 支持离线使用吗？

A: 桌面版本身支持离线运行，但 LLM 服务需要网络连接。如果使用本地部署的 LLM（如 Ollama），则可完全离线使用。

Q: 如何更新到最新版本？

A:

  • 桌面版: 启动时会自动检查更新，或手动下载最新安装包
  • Web 版: 自动使用最新版本
